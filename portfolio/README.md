## ğŸ¨ Customization

### Colors
The main color scheme uses CSS custom properties. To change colors, modify these values in `style.css`:
- Primary gradient: `#667eea` to `#764ba2`
- Accent colors: `#00f5ff` (cyan) and `#ff00ff` (magenta)

### Content
- **Personal Info**: Update in the About section of `index.html`
- **Projects**: Modify project cards with your actual project details
- **Skills**: Adjust skill percentages in the `data-width` attributes
- **Social Links**: Replace placeholder URLs with your actual profiles

### Fonts
Currently uses Google Fonts (Poppins). To change:
1. Update the Google Fonts link in `<head>`
2. Modify `font-family` in CSS
