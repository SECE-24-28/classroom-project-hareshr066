# MyCourseHub - Mini Udemy Website

A simple, modern course learning platform built with HTML, CSS, and JavaScript.

## Features

- **Modern UI Design**: Glass-morphism login page with purple gradient theme
- **Responsive Layout**: Works on desktop and mobile devices
- **Course Catalog**: 4 sample courses with preview functionality
- **Video Modal**: YouTube video integration for course previews
- **Simple Navigation**: Clean navigation between pages

## File Structure

```
project/
├── index.html        # Home page with courses
├── login.html        # Login page with glass-morphism design
├── styles.css        # All styling and responsive design
├── script.js         # JavaScript functionality
└── assets/
    └── images/       # Course thumbnail placeholders
```

## How to Use

1. Open `login.html` in your browser
2. Click "Login" button (no credentials required)
3. Browse courses on the home page
4. Click "Watch Preview" to see YouTube video modals

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Gradients)
- Vanilla JavaScript
- SVG placeholders for course images

- Purple gradient color scheme (#6a11cb → #2575fc)
